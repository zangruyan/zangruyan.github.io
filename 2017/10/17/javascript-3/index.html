数组

  • 数组[] Array是一种数据格式。
  • 数组中，每个数据之间用“,”隔开。
  • 获取数组中的其中一位，需要用下标。
  • 数组的长度 length。
  • 数组中可以存放任意类型的数据,但是一般为了代码的可读性极维护性，我每一个数组中，只存放一种类型的数据。
  • 当判断不大于这个数组的时候可以两种方式。
    • i>=arr.length-1;
    • i>arr.length
